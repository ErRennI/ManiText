# ManiText

- A text editor with garbage-collection

## Requirements

- GCC
- GNU make
- ncurses library

## Installing Dependencies

**Ubuntu/Debian:**

```bash
sudo apt install gcc make libncurses5-dev libncursesw5-dev
```

**Fedora:**

```bash
sudo dnf install gcc make ncurses-devel
```

**Arch Linux:**

```bash
sudo pacman -S gcc make ncurses
```
## Build

```bash
make
```

## Run

```bash
make run
```

## Clean

```bash
make clean
```
